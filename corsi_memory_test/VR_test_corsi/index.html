Corsi Memory Test Tutorial

Instructions

Welcome to the Corsi Memory Test! In this test, you will see a sequence of blocks being highlighted one by one. Your goal is to memorize and reproduce this sequence by clicking the blocks in the same order. The following video is a visual illustration of what we mean:

The game starts with a sequence length of two, and the sequence length will increase by one for each round you complete correctly. If you make two mistakes in a row, the game is over. Your final score is the length of the longest sequence you successfully remember.
